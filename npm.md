除了上传至 npm 来使用包之外，我们还能够借助 github，直接通过github项目安装：npm i git+https://github.com/MrEnvision/XXX.git

{% hint style="warning" %}
建议 github 项目使用 tag 来进行版本号管理，安装的时候加上版本号，npm i git+https://github.com/MrEnvision/XXX.git\#版本号
{% endhint %}
