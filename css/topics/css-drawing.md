## 3. border-radius属性

✍️ 扇形画法是利用 border-radius 并设置透明度来实现的，类似于 CSS 画三角形。

![](./img/circle.png)

border-radius 可以用来实现圆形，这个众所周知，但是一个鲜为人知的内容是，border-radius 接受水平和垂直方向不同值，使用斜杠(/)来分隔它们，这可以让我们在圆角处取整来创建椭圆。

<img src="./img/tuoyuan.png" style="zoom:90%;" />
