# 模块化

> 许多语言都具有模块支持，例如 python 可通过 import 导入模块，CSS 可通过 @import 导入模块，但是 JavaScript 语言没有模块支持，即不同 JS 文件之间不能直接通过诸如 import 这样的方式进行导入使用，本文主要介绍了JavaScript 文件进行模块化的几种方式。
